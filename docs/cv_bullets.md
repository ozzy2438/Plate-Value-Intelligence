# CV Bullets

- Built an explainable licence-plate valuation and portfolio-routing system from 17,782 governed DVLA auction records, using event-based validation and an untouched final holdout.
- Developed Ridge, Random Forest and Gradient Boosting valuation models, achieving approximately GBP 1.25K final-test MAE and GBP 621 median absolute error while capturing 55% of top-decile premium assets.
- Designed a transparent premium and collectibility framework identifying 507 collectible-potential registrations and routing approximately 10% of the portfolio to specialist human review.
- Converted valuation and rarity signals into fixed-price, standard-auction, premium-showcase and specialist-review recommendations for 1,981 final-event assets, with governed reserve-policy and sensitivity simulations.
